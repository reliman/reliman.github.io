To customize the template you can edit CSS and HTML files. Also if you are a developer, there are available source files SCSS, JS (ES6) and HTML (Nunjucks). All used plugins added from npm.

Show on Scroll (Custom Script)

A lightweight and simple script performs the action of appearing an element during page scrolling.

  • Duration (milliseconds) - data-show-duration="500"
  • Delay (milliseconds) - data-show-delay="100"
  • Distance (px) - data-show-distance="10"
  • Position (default: bottom) - data-show-origin="left" - top | right | bottom | left
