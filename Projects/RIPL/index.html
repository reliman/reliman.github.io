RIPL (Research Institute for Public Libraries)

About Project

The ALA Emerging Leaders 2018 group was in need of developing a data competencies training. Our research group found that the best way to reach a larger audience would be to create an online web resource-based toolkit for individuals interested in learning about best practices in data collection, management, and analysis.

Methods:

A/B Tests and Card Sorting

A/B Tests: These were used to test different website features such as home page features (Carousel and data-placement survey). By using A/B Tests I was able to narrow down which features to use and add to the Data Competencies website.

Card Sorting Tests: This was used to help create a usable navigability system for the website. Since this website was/is used to help folks get started with learning how to collect data and analyze data, we wanted to make sure that users of all technology aptitudes were able to use this resource.
